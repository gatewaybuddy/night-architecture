## Quick Import Instructions

### Method 1: Google Calendar Web (Recommended)

1. **Open Google Calendar**
   - Go to [calendar.google.com](https://calendar.google.com)
   - Make sure you're logged into your account

2. **Access Settings**
   - Click the gear icon (⚙️) in the top right
   - Select "Settings" from the dropdown

3. **Import Calendar**
   - In the left sidebar, click "Import & Export"
   - Click "Select file from your computer"
   - Navigate to: `/mnt/d/projects/.claude-organizer/calendar-import.ics`
   - Select the file

4. **Choose Calendar**
   - Select which calendar to add events to (use your primary calendar or create a new "Claude Projects" calendar)
   - Click "Import"

5. **Verify Import**
   - Go back to your calendar view
   - You should see events starting January 27, 2026
   - Events are color-coded by category

### Method 2: Create New Calendar First (Better Organization)

1. **Create Dedicated Calendar**
   - In Google Calendar sidebar, click the "+" next to "Other calendars"
   - Select "Create new calendar"
   - Name: "Rado's Claude Projects"
   - Description: "Strategic project management for active development projects"
   - Click "Create calendar"

2. **Import Into New Calendar**
   - Follow steps 2-5 from Method 1 above
   - When choosing calendar, select "Rado's Claude Projects"

## Calendar Management Tips

### Customize as Needed

**Adjust Times:**
- All times are set to UTC
- Adjust to your local timezone and preferred work hours
- Move events to fit your schedule

**Set Reminders:**
1. Click any event
2. Click "Edit event"
3. Add notifications (e.g., 1 hour before, 1 day before)
4. Save

**Color Coding:**
- Customize colors for different priorities
- HIGH priority = Red
- MEDIUM priority = Orange
- LOW priority = Yellow

**Add Recurring Events:**
- Weekly reviews are already recurring (every Monday)
- Monthly reviews are already recurring (last Friday)

### Track Progress

**Mark Tasks Complete:**
1. When you finish a task, click the event
2. Change the title to add ✓ at the beginning
3. Example: `✓ [CLAUDE] AUTOSPOT - Review codebase for bugs`

**Add Notes:**
- Click any event to add notes about progress
- Document blockers or issues
- Link to relevant files or commits

**Share Calendar:**
- Share "Rado's Claude Projects" calendar with team members if needed
- They can view progress and milestones

## Troubleshooting

### Events Not Showing?
- Check calendar date range (should show Jan-Apr 2026)
- Verify calendar visibility (make sure calendar is checked in sidebar)
- Refresh page or sync calendar

### Wrong Timezone?
- Google Calendar auto-adjusts to your timezone
- If times seem off, check your Google Calendar timezone settings
- Settings > General > Time zone

### Duplicate Events?
- If you import twice, you'll get duplicates
- Solution: Delete the calendar and reimport
- Or manually delete duplicate events

### Can't Find Import Option?
- Make sure you're using Google Calendar web version
- Some mobile apps don't support ICS import directly
- Use desktop web browser instead
